= Generate marvelous aerial view of your project !

image:https://img.shields.io/maven-central/v/io.github.jtama/project-graph-generator.svg?label=Maven%20Central[Maven Central, link=https://search.maven.org/artifact/io.github.jtama/project-graph-generator] image:https://img.shields.io/badge/License-Apache%202.0-blue.svg[License,https://opensource.org/licenses/Apache-2.0] image:https://github.com/jtama/project-graph-generator/workflows/EarlyAccess/badge.svg[Build, link=https://github.com/jtama/project-graph-generator/actions?query=workflow%3AEarlyAccess]

The recipe will try to autodetect the base package of your project.

== Available options

[horizontal]
`maxNodes`:: The maximum number of nodes in the final graph. Will drop the nodes with less weight
`basePackages`:: A list of comma separated package names which contain the targeted classes.

== How to execute

With default options

[source,console]
----
mvn -U org.openrewrite.maven:rewrite-maven-plugin:run \
-Drewrite.recipeArtifactCoordinates=io.github.jtama:project-graph-generator:1.0.0 \
-Drewrite.activeRecipes=io.github.jtama.openrewrite.ProjectAerialViewGenerator
----

With full options

[source,console]
----
mvn -U org.openrewrite.maven:rewrite-maven-plugin:run \
-Drewrite.recipeArtifactCoordinates=io.github.jtama:project-graph-generator:1.0.0 \
-Drewrite.activeRecipes=io.github.jtama.openrewrite.ProjectAerialViewGenerator \
-Drewrite.options=maxNodes=8,basePackages=com.foo
----
